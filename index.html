1. Presentación de la actividad

En esta simulación se modela el aumento de temperatura en un sistema calorimétrico a partir de la combustión de un combustible. El estudiante debe ingresar los datos experimentales, ejecutar la simulación, analizar la tabla y la gráfica, completar el informe y finalmente descargar el reporte en PDF con toda la información.

3. Instrucciones para el estudiante

  1. Complete sus datos personales.
  2. Configure el experimento con los valores indicados.
  3. Ejecute la simulación.
  4. Revise la tabla de temperaturas generadas por segundo.
  5. Analice la gráfica de temperatura vs. tiempo.
  6. Complete todos los apartados del informe.
  7. Descargue el informe en PDF.

Nota: esta herramienta tiene fines didácticos. Los valores energéticos son aproximados y se emplean para modelación educativa.

4. Configuración del experimento

Modelo utilizado

Q = m · poder energético
C = Ccal + (cagua · magua)
ΔT = Q / C

Luego, el incremento térmico se distribuye segundo a segundo para generar la curva de temperatura.
